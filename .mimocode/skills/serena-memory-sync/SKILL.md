# serena-memory-sync

Serena memories are durable cross-tool source-of-truth facts. MiMoCode
`MEMORY.md` is a runtime-native projection from verified facts.

Only store:

- Paths and entry points.
- Contracts and invariants.
- Update triggers.
- Verification commands.

Do not store chat logs, speculation, secrets, cookies, tokens, or raw runtime
state.
