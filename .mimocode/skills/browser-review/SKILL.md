# browser-review

Route browser work through the approved providers:

- Webwright: long-horizon browser workflows and reusable evidence scripts.
- Playwright CLI: deterministic screenshots, snapshots, traces, and visual
  evidence.
- Chrome DevTools MCP: console, network, performance, memory, Lighthouse, and
  live Chrome debugging.

Do not introduce unapproved browser providers without a policy update and
validator.
